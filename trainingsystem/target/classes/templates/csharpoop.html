Основы ООП

Объектно-ориентированное программирование (ООП) в C# основано на четырех принципах: инкапсуляция, наследование, полиморфизм и абстракция.

Инкапсуляция

Инкапсуляция скрывает внутренние детали объекта и предоставляет только необходимые методы для взаимодействия с ним.

            
class Person {
    private string name;
    private int age;

    public string Name {
        get { return name; }
        set { name = value; }
    }

    public int Age {
        get { return age; }
        set { age = value; }
    }
}

Наследование

Наследование позволяет создавать новые классы на основе существующих.

            
class Employee : Person {
    public string Company { get; set; }
}

Полиморфизм

Полиморфизм позволяет использовать объекты производных классов через интерфейсы или базовые классы.

            
class Program {
    static void Main(string[] args) {
        Person person = new Employee();
    }
}

Абстракция

Абстракция позволяет создавать абстрактные классы и методы, которые могут быть реализованы в производных классах.

            
abstract class Animal {
    public abstract void MakeSound();
}

class Dog : Animal {
    public override void MakeSound() {
        Console.WriteLine("Woof");
    }
}
