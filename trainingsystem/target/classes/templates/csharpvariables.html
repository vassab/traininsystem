Переменные и типы данных

В C# существует несколько типов данных, включая целые числа, числа с плавающей точкой, строки и логические значения. Переменные используются для хранения данных.

            
int myInt = 5;
float myFloat = 5.75f;
string myString = "Hello, World!";
bool myBool = true;
