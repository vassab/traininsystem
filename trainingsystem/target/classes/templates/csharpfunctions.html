Функции и методы

Функции и методы используются для выполнения определенных задач. В C# методы объявляются внутри классов.

            
class Program {
    static void Main(string[] args) {
        MyMethod();
    }

    static void MyMethod() {
        Console.WriteLine("Hello from a method!");
    }
}
