Управляющие конструкции

Управляющие конструкции в C# включают в себя условные операторы (if, switch) и циклы (for, while, do-while).

            
if (myBool) {
    Console.WriteLine("True");
} else {
    Console.WriteLine("False");
}

for (int i = 0; i < 10; i++) {
    Console.WriteLine(i);
}
