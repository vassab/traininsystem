Управляющие конструкции

В C++ доступны различные управляющие конструкции для управления потоком выполнения программы:

  • if и else - для условных операторов
  • for - для циклов с известным числом итераций
  • while и do-while - для циклов с неизвестным числом итераций
  • switch - для многократного выбора
