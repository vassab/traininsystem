Переменные и типы данных

В C++ переменные используются для хранения данных различных типов. Основные типы данных включают:

  • int - для целых чисел
  • float - для чисел с плавающей запятой
  • double - для чисел с двойной точностью
  • char - для символов
  • bool - для булевых значений
